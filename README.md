## Commands

**Start Preview**

```console
npm run dev
```

**Render still image**

```console
npm run render
```

**Upgrade Remotion**

```console
npm run upgrade
```

## Server

You can run a server that serves dynamic images based on it's URL. Run

```console
npm run server
```

And then visit `http://localhost:8000/PreviewCard.png?title=Hello+World` in your browser to render an image.
